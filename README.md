# Discord-Record-Bot
**This is an unfinished bot**  
Mixer by [Licson](https://github.com/licson0729)  
Record discord voice channel and upload files(.mp3) to telegram

## How to use
```
1. npm install 
2. Edit config.json.example and rename it config.json
3. npm run start
```
